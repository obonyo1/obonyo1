I'm Paul Obonyo, a Computer Science student and software engineer building machine learning systems, cloud applications, and intelligent software.
I enjoy designing scalable software that solves meaningful real-world problems, particularly through ML, distributed systems, and secure application development.

## 🔬 Current Focus

### Swahili Speech-based Concern Classification & Routing

My undergraduate project explores how speech recognition, transformer-based language models, and explainable machine learning can automatically classify spoken concerns and route them to the appropriate community organizations serving Nairobi's informal settlements.

---
### Learning & Exploration
  - ☁️ Cloud Computing — core services, architecture, and deployment models
  - 🔒 Cloud Security — securing infrastructure and data in cloud environments
  - 🔁 CI/CD — automating build, test, and deployment pipelines
  - 🗣️ Transformer-based NLP for low-resource African languages
  - 🔍 Explainable ML (SHAP) and real-time dashboard architectures

### Core Interests
- Social impact technology, Human-Computer Interaction, NLP for African languages, embedded systems and IoT, applied machine learning, cybersecurity, and multimedia/web development.
